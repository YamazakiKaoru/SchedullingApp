# 機能一覧
 ### ●ログイン・ログアウト<br>
 ### ●シフト申請<br>
 ### ●プロフィールの編集<br>
 ### ●画像のアップロード<br>

# 技術
 ### セッション情報
 　SpringSecurityでログイン情報の管理<br>
 ### シフト管理
 　postgresqlに保存し表示<br>
   SpringDataJPAでの実装
 ### 求人情報
 Yahooディペロッパーネットワークの求人情報取得APIの実装<br>
 　- https://developer.yahoo.co.jp/webapi/job/v1/jobinfo.html
